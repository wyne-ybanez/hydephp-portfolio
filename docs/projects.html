Projects

In this collection you'll find a diverse array of milestone projects, largely developed out of personal interest. Feel free to contribute or clone any of these.

Digi.Art - Custom Laravel E-commerce

This is a Laravel project for an e-commerce website/app that allows online sellers to sell, categorize and take payment for the sale of goods online. I made it for a fictional digital art shop but can be used to scaffold a basic e-commerce site.

WeatherTop V2

Version 2.0 of WeatherTop project. A weather station reporting and reading app. Utilizes the OpenWeather API. Made with the Express JS framework.

Sneak'r Heads

Sneak'r Heads is a Shoe E-Commerce Store. It's a project that uses Django for full stack development and is one of my milestone projects.

Space Invaderz Ultra

Space invaders built on canvas using JavaScript. A classic spaceship arcade game created through Object Oriented Programming, game styling and particle effects.

Scrape Deep

A web content scraper made using PHP, this is a Scrapeheap replica but improved further with user-interfaces and different file formats. Used to scrape content from an obsolete site.

Old Portfolio Site

A User-Centric Portfolio project used to display what I had learned whilst first learning how to code. This was my very first milestone project. Made using foundational HTML and CSS skills.
